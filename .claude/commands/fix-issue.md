# /fix-issue — Bug Fix Workflow

Fix a reported issue with minimal, targeted changes.

## Arguments
- Issue description or number

## Steps

1. Understand the issue — read related code, reproduce if possible
2. Identify root cause — trace the bug to its source
3. Apply minimal fix — change only what's necessary
4. Verify the fix — rebuild and test
5. Check for regressions — ensure no other behavior changed
6. Summarize what was wrong and what was changed
